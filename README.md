# BlokPOST - Система управленческой отчетности

Веб-приложение для управленческой отчетности в сети магазинов спецодежды «БлокПОСТ». Объединяет разрозненные данные из Excel в единую версию правды.

## Архитектура

- **Фронтенд**: Next.js с TypeScript, Tailwind CSS
- **Бэкенд**: Next.js API Routes
- **База данных**: PostgreSQL с Prisma ORM
- **Авторизация**: NextAuth.js с ролевой моделью
- **ETL**: Обработка Excel файлов с помощью xlsx

## Сущности

- Магазины
- Товары (с категориями)
- Продажи
- Остатки
- Справочник KPI
- Пользователи с ролями: Собственник, Аналитик, Директор магазина

## Запуск

1. Установите зависимости: `npm install`
2. Настройте базу данных PostgreSQL и обновите `.env`
3. Запустите миграции: `npx prisma migrate dev`
4. Запустите сервер: `npm run dev`
